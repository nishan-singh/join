What is Join?

Join is a new project management tool which is based on the Kanban system.
With the Kanban system everyone of your team can see which task: - has to be done, - is in progress, - is awaiting for Feedback, - is done.

How to use it

  1. Create a new task by clicking on "Add Task". There you have to fill in the fields (Subtasks are optional)
  2. On the Board you can see all your tasks and you are also able to move them from one section to the other
  3. All important infomation of the contacts are displayed in the "contact"-section. You also have the opportunity to add new contacts.
